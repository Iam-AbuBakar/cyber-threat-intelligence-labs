# Cyber Threat Intelligence (CTI) Hands-On Labs

This repository contains hands-on Cyber Threat Intelligence (CTI) labs performed using industry-standard threat intelligence platforms.

## Tools Covered
- VirusTotal
- Shodan
- URLhaus
- GreyNoise
- SOCRadar (IOC, CVE, Threat Actor)
- AlienVault OTX
- Cisco Talos

## Objectives
- Investigate malicious indicators (IP, domain, URL, hash)
- Correlate indicators with threat intelligence sources
- Map findings to MITRE ATT&CK
- Perform SOC-level risk assessment and response decisions

## Skills Demonstrated
- IOC enrichment
- Threat correlation
- CVE analysis
- Threat actor profiling
- SOC alert triage
- CTI documentation
